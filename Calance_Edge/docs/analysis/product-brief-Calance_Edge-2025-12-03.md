**Author:** JB

## Executive Summary

Calance Edge represents a transformative shift in sales operations, empowering Calance's sales team to create impactful, high-quality artifacts with consistent excellence through an intuitive AI-powered interface. By consolidating three essential workflows—Case Studies, Presentations, and Recruiting Toolkit—into a unified platform, Calance Edge eliminates fragmented manual efforts and replaces them with AI-driven precision and professional quality.

Positioned for the new year, Calance Edge serves as both a tactical efficiency tool and a strategic demonstration of AI's potential within Calance's operations. The platform embodies the principle that "sales artifacts are Edged"—crafted with AI precision while maintaining authentic Calance quality and brand consistency.

---

## Core Vision

### Problem Statement

Calance's sales team currently operates with fragmented, manual workflows using Microsoft Office tools and SharePoint for creating critical sales artifacts, resulting in inconsistent quality, wasted time, and missed opportunities to leverage AI's transformative potential. The team lacks awareness of and access to AI tools that could dramatically enhance their operational effectiveness.

### Problem Impact

The manual, "dis-combobulated" approach to sales artifact creation using traditional Office tools leads to significant operational inefficiencies, inconsistent branding, and lost competitive advantage. As the new year approaches, the sales team needs a unified solution that accelerates momentum while elevating the quality of all customer-facing materials.

### Why Existing Solutions Fall Short

Current approaches rely on disconnected Microsoft Office tools (PowerPoint, Word, Excel) and SharePoint that don't leverage AI capabilities, require significant time investment, and lack brand consistency. While AI tools exist, they present learning curves and adoption barriers that prevent effective utilization by sales teams focused on revenue generation.

### Proposed Solution

Calance Edge is a unified web application (React + Vite frontend, Flask backend) that consolidates sales enablement tools into a cohesive, branded experience. The platform combines three core modules—Case Study Edge, Presentation Edge, and Recruiting Toolkit—each designed with zero-training interfaces that transform hours of manual work into minutes of AI-assisted creation. The solution emphasizes quality improvement and team empowerment, enabling consistent excellence across all sales operations artifacts.

**Technical Implementation:** Built with Docker containerization, OpenRouter API integration, and intelligent model routing (Gemini Flash for drafts, Claude Sonnet for final polish, GPT-4o Mini for structured extraction), Calance Edge delivers professional outputs in multiple formats including downloadable PDFs, standalone HTML presentations, and copy-to-clipboard text artifacts.

### Key Differentiators

- **Unified Toolbox Philosophy**: Integrated ecosystem where complementary workflows (Case Studies → Presentations → Recruiting) work together seamlessly
- **Zero-Training Interface**: Intuitive design that enables immediate adoption without training overhead (< 2 minute onboarding)
- **Quality at Scale**: AI-powered consistency with success metrics including < 5 minutes for case studies and < 10 minutes for presentations
- **Brand Perfection**: Automatic Calance styling applied to all outputs, ensuring consistent excellence
- **Strategic Timing**: Positioned for new year momentum, giving the sales team an immediate competitive advantage
- **Technical Excellence**: Modern stack (React, Flask, Docker) with intelligent LLM routing and multi-format export capabilities

## MVP Scope

### Core Features

**Case Study Edge Module:**
- Complete input form with all required fields (client_name, industry, title, challenge, solution, tech_stack, metrics, benefits, ROI)
- LLM-powered draft generation using intelligent routing (Gemini Flash for drafts, Claude Sonnet for quality refinement)
- Live preview system with feedback → regenerate workflow
- Dual export capabilities: PDF generation via ReportLab and standalone HTML via Jinja2
- Automatic Calance branding and styling application

**Presentation Edge Module:**
- Presentation wizard with title, purpose, audience, and theme selection interface
- Key points input system that converts talking points into professional slides
- Multiple slide templates: title, section, content, two-column, quote, metrics, closing layouts
- Slide preview carousel with navigation and per-slide selection
- Per-slide feedback system with targeted regeneration capabilities
- Standalone HTML export with embedded keyboard navigation and responsive design

**Recruiting Toolkit Module:**
- Tabbed interface with all 8 recruiting tools fully operational
- Core functionality: JD Enhancer, Sourcing Email Generator, Boolean Search String Creator, Candidate Submittal Generator, Interview Prep Email Builder, Mock Interview Questions, Skills/Title/Location Extractor, Executive Summary Writer
- Copy-to-clipboard functionality for all generated outputs
- Professional markdown formatting for all recruiting artifacts

**Shared Infrastructure:**
- React + Vite frontend with unified navigation shell and brand-consistent UI components
- Flask backend with Gunicorn WSGI server and comprehensive API endpoints
- OpenRouter API integration with intelligent LLM routing system
- Docker containerization with nginx reverse proxy for scalable deployment
- Brand engine with Calance styling system and asset management

### Out of Scope for MVP

- **PPTX export functionality** from Presentation Edge (planned for Phase 2)
- **Nano-Banana image generation** integration (planned for Phase 2)
- **Advanced analytics dashboard** beyond basic usage tracking
- **Mobile application version** (focus on responsive web interface)
- **External CRM or sales tool integration** (future enhancement)
- **Advanced user management** and role-based access control
- **Content versioning and collaboration** features
